# hashsum by Golang

> 使用说明

```
Usage of hashsum.exe, Version: 1.0.1
hash sum file/string by K.o.s[vbz276@gmail.com]!
  -crc32
        use CRC32 hash
  -crc64
        use CRC64 hash
  -f string
        to Hash file
  -md5
        use MD5 hash
  -s string
        to Hash string
  -sha1
        use SHA1 hash
  -sha256
        use SHA256 hash
  -sha512
        use SHA512 hash
```



> Example
```
hashsum.exe -f "D:\zh-hans_windows_xp_professional_with_service_pack_3_x86_cd_vl_x14-74070.iso" -md5 -sha1 -crc32
```
